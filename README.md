# Canny Carpentry

OC Carpentry business with Landing page, contact us, and e-commerce section.

## Overview

This is a Vue.js front-end project for Canny Carpentry, featuring:

- **Landing Page**: Showcase of the business and services
- **Contact Us**: Get in touch form for customer inquiries
- **E-Commerce Section**: Browse and purchase carpentry products and services

## Tech Stack

- **Frontend Framework**: Vue.js
- **Language**: JavaScript
