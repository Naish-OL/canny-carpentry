## Design System

1. **Gray** - `#A8A8A8` (secondary backgrounds, dividers)
   - Text: #333333 or #1a1a1a (dark gray/charcoal)

2. **Sage Green** - `#99B8A3` (primary sections, feature areas)
   - Text: #333333 or #1a1a1a (dark gray/charcoal)

3. **Warm Beige** - `#F5E9E5` (light backgrounds, information sections)
   - Text: #333333 or #1a1a1a (dark gray/charcoal)

**Cards & Content (E-Commerce & Landing):**

**Interactive Elements:**
- `Button` - All buttons

### When Providing Code Suggestions:

1. **Import PrimeVue components** at the top of every component
   ```javascript
   import { ref } from 'vue'
   import Button from 'primevue/button'
   import Toolbar from 'primevue/toolbar'
   import Card from 'primevue/card'
   ```

2. **Use Tailwind CSS classes** for styling alongside PrimeVue
   - Use `p-4`, `m-4`, `mt-6`, etc. for padding/margin
   - Use `bg-cki-sage`, `bg-cki-beige`, `bg-cki-gray` for backgrounds
   - Use `text-cki-dark` for text
   - Use responsive classes: `md:`, `lg:`, `sm:` prefixes

3. **Example PrimeVue Component:**
   ```vue
   <template>
     <Card class="m-4 bg-cki-beige shadow-lg">
       <template #header>
         <h2 class="text-2xl font-bold text-cki-dark">Section Title</h2>
       </template>
       <p class="text-cki-dark">Content goes here</p>
     </Card>
   </template>
   ```

4. **Responsive Design**
   ```vue
   <div class="flex flex-col md:flex-row gap-4 p-4 md:p-8">
     <div class="w-full md:w-1/2">Left content</div>
     <div class="w-full md:w-1/2">Right content</div>
   </div>
   ```

### Responsive Breakpoints

- **Mobile:** < 768px (sm)
- **Tablet:** 768px - 1024px (md)
- **Desktop:** > 1024px (lg, xl)

Use Tailwind's responsive prefixes: `sm:`, `md:`, `lg:`, `xl:`

## Vue Router Configuration

Create `/src/router.js`:

```javascript
import { createRouter, createWebHistory } from 'vue-router'
import Home from './pages/Home.vue'
import Shop from './pages/Shop.vue'
import About from './pages/About.vue'
import Gallery from './pages/Gallery.vue'
import Contact from './pages/Contact.vue'
import ProductDetail from './pages/ProductDetail.vue'

const routes = [
  { path: '/', component: Home, name: 'Home' },
  { path: '/shop', component: Shop, name: 'Shop' },
  { path: '/about', component: About, name: 'About' },
  { path: '/gallery', component: Gallery, name: 'Gallery' },
  { path: '/contact', component: Contact, name: 'Contact' },
  { path: '/product/:id', component: ProductDetail, name: 'ProductDetail' },
]

export default createRouter({
  history: createWebHistory(),
  routes
})
```

8. **Error handling** - Display user-friendly error messages
