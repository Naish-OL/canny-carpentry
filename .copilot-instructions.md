# Copilot Instructions for Canny Carpentry

## Project Overview

This is a **Vue.js front-end project** for Canny Carpentry, a carpentry business website with the following features:

- Landing page
- Contact form/section
- E-commerce functionality

## Tech Stack

- **Primary Framework**: Vue.js 3 (JavaScript)
- **UI Component Library**: PrimeVue (Modern Vue 3 components)
- **Styling**: Tailwind CSS
- **Icons**: PrimeIcons
- **Project Type**: Single Page Application (SPA)

## Key Guidelines for Copilot

### **IMPORTANT: PrimeVue Usage**

**ALL UI components must use PrimeVue components.** Do not use custom HTML or unstyled elements for layout and interactive components.

#### Chakra UI Components to Use:

**Layout & Structure:**
- `CBox` - Container and layout wrapper
- `CContainer` - Responsive container
- `CFlex` - Flexbox layout
- `CGrid` - Grid layout
- `CStack` (HStack, VStack) - Stack layouts

**Navigation & Headers:**
- `CBox` or `CFlex` for navigation bars
- `CButton` for navigation links/CTAs

**Forms (Contact Section):**
- `CInput` - Text inputs
- `CTextarea` - Textarea fields

- `CFormControl` - Form wrapper
- `CFormLabel` - Form labels
- `CFormErrorMessage` - Error handling

**Cards & Content (E-Commerce & Landing):**
- `CBox` - Card containers

- `CHeading` - Headings (h1, h2, h3, etc.)

- `CButton` - All buttons

### When Providing Code Suggestions:

1. **Import Chakra UI components** at the top of every component
   ```javascript
   import { CBox, CContainer, CFlex, CButton, CHeading, CText, CImage } from '@chakra-ui/vue'
   ```

2. **Use Chakra UI props for styling** instead of CSS classes
   - Use `m`, `p`, `mt`, `mb`, `ml`, `mr` for margins/padding
   - Use `w`, `h` for width/height
   - Use `bg`, `color`, `fontSize` for styling
   - Use `display`, `alignItems`, `justifyContent` for layout

3. **Example Chakra UI Component:**
   ```vue
   <template>
     <CBox bg="gray.50" p="4" borderRadius="md">
       <CHeading as="h2" size="lg" mb="2">
         Section Title
       </CHeading>
       <CText color="gray.600">
         This is a sample component using Chakra UI.
       </CText>
     </CBox>
   </template>
   ```

4. **Use Chakra UI spacing system** - Use consistent spacing: `2`, `4`, `6`, `8`, `12`, `16`, `20`, etc.

5. **Responsive design with Chakra UI** - Use array syntax for responsive values:
   ```vue
   <CFlex 
     direction={["column", "row"]} 
     spacing={["2", "4"]} 
     p={["4", "8"]}
   >
     <!-- Content -->
   </CFlex>
   ```

### Component Structure

- **Landing Page Component** (`/src/components/LandingPage.vue`)
  - Hero section with CTA button
  - Features/services showcase using CGrid
  - Call-to-action section

- **Contact Section** (`/src/components/ContactSection.vue`)
  - Contact form with Chakra UI form components
  - Contact information display
  - Success/error messaging

- **E-Commerce Section** (`/src/components/ECommerceSection.vue`)
  - Product grid using CGrid
  - Product cards using CBox
  - Shopping cart interface
  - Product filtering (if applicable)

- **Navigation Component** (`/src/components/Navigation.vue`)
  - Header with CFlex layout
  - Navigation links
  - Mobile menu using CDrawer (if needed)

### When Creating New Features:

1. Always prioritize Chakra UI components first
2. Use Chakra UI theme colors and spacing
3. Ensure responsive design using Chakra UI's responsive array syntax
4. Never mix Chakra UI with custom CSS for the same component
5. Use Chakra UI's built-in accessibility features
